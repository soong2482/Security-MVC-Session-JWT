
# Security Session+JWT

기초적인 Spring Security를 사용한 MVC 프로젝트입니다. 이 프로젝트에서는 회원가입(SignUp), 로그인(Login), 로그아웃(Logout), 그리고 Role 권한에 따른 API 요청을 구현합니다
